# **App Name**: OmniCalc

## Core Features:

- Diverse Calculator Library: Offer a wide range of calculators in categories such as finance, health, math, fitness, and construction.
- Client-Side Calculations: All calculations performed instantly on the client-side for immediate results.
- Explanation Display: Each calculator provides a clear, concise explanation of the calculation method.
- Embed and Share Features: Enable users to embed calculators on other websites and share them easily.
- User Accounts: Allow users to register and manage accounts for saving calculations and preferences.
- Saved Calculations: Users can save their calculations to their accounts for future reference.
- Analytics Tracking: Gather anonymous data on calculator usage to provide tool suggestions.

## Style Guidelines:

- Primary color: A calm, sky-inspired blue (#63B3ED) to evoke trust and reliability.
- Background color: Light gray (#F7FAFC) to ensure readability and a clean interface.
- Accent color: A gentle, harmonious green (#48BB78) to signal success and accuracy in calculations.
- Body and headline font: 'Inter' for a modern, neutral, readable style, suitable for all content.
- lucide-react icons should be used. These should match the theme and be used for various calculator features
- Use a grid-based layout for calculator categories and individual calculators.
- Subtle Framer Motion animations for transitions between calculators and changes in numerical values.
